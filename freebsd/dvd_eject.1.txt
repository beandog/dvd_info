DVD_EJECT(1)
============
:doctype: manpage

NAME
----
dvd_eject - eject or close an optical device tray

SYNOPSIS
--------
*dvd_eject* ['OPTIONS'] ['DEVICE']

DESCRIPTION
-----------
Use FreeBSD's 'cdcontrol' program to eject or close an optical tray, adding DVD-specific options.

OPTIONS
-------

*-t*
	Close tray instead of opening.

*-d*
	Display DVD id after closing tray using dvd_info.
